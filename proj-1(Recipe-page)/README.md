### What I learned
In this project, I reinforced my understanding of professional CSS standards:

1. **The Centering Rule:** To center an element using `margin: auto`, it must have `display: block` and a defined `width`.
2. **Logical Ordering:** I learned to always follow a proper element order in CSS (Dimensions first, then Colors, and finally Typography).
3. **List Styling:** Using the `::marker` selector to style list bullets independently from the text.
4. **Tables:** Using `:last-child` to remove the bottom border from the final table row for a clean look.
