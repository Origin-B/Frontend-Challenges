This is my solution to the [Blog Preview Card challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/blog-preview-card-ckPaj01IcS).

## 🚀 The Challenge

The goal was to build a responsive blog preview card and get it looking as close to the design as possible.

### ✨ Features
* **Semantic HTML5:** Using proper tags like `<header>`, `<main>`, and `<footer>`.
* **CSS Grid Layout:** For precise control over the card's internal spacing.
* **Neo-brutalism Design:** Implementing sharp shadows and thick borders.
* **Responsive Design:** Optimized for both mobile and desktop screens.
* **Custom Branding:** Integrated my official **OriginB** logo in the footer.

## 🛠️ Built With
* Semantic HTML5 markup.
* CSS Custom Properties (Variables).
* CSS Grid & Flex-box.
* Google Fonts (Figtree).

## 📏 My Golden Rules (Applied)
During this project, I strictly followed my coding principles:
1. **Centering Mastery:** To center an element using `margin: auto`, it must have `display: block` and a defined `width`.
2. **CSS Ordering:** Followed a strict property order (Dimensions first, then Colors/Styles) to keep the code clean and professional.
