# Bento Grid — Frontend Mentor Challenge

![Project Screenshot](./design/screenshot.png)

A solution to the [Bento Grid challenge](https://www.frontendmentor.io/challenges/bento-grid-RMydElrlOj) on Frontend Mentor.

---

## Overview

A responsive Bento Grid layout built with pure HTML & CSS. The layout shifts from a single-column mobile view to a 4-column desktop grid using CSS Grid.

---

## Built With

- Semantic HTML5
- CSS Custom Properties
- CSS Flexbox
- CSS Grid
- Mobile-first workflow

---

## What I Learned

- Placing grid items manually using `grid-column` and `grid-row`
- Using `overflow: hidden` to clip images inside grid cards
- Building a responsive layout without any frameworks
